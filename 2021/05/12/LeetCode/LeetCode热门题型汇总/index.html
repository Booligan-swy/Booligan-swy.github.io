1.两数之和（简单）

题目描述：

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

题解思路：

第一种解法是直接暴力穷举，但是这样很耗时间，因此考虑第二种解法，利用hash表。

首先遍历一次数组，以数组中的元素作为hash表的下标，然后索引+1作为hash表对应的值。然后再对数组进行一次遍历，考察$target-nums[i]$在hash表中是否存在，并且要考虑剔除重复选中自身的情况。

实现代码：

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
       unordered_map<int, int> hs;
       int n = nums.size();
       //记录nums数组中的元素
       for(int i = 0; i < n; i++){
           hs[nums[i]] = i+1;
       }

        vector<int> result;
       for(int i = 0; i < n; i++){        
            if(hs.count(target-nums[i]) > 0 && hs[target-nums[i]] - 1 > i){
                result.emplace_back(i);
                result.emplace_back(hs[target-nums[i]]-1);
                break;
            }
       }

       return result;
    }
};

代码的复杂度分析：

  • 时间复杂度：O(n)，n是数组的长度；
  • 空间复杂度：O(n)，空间消耗主要来源于hash表。

20.有效的括号（简单）

题目描述：

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。

题解思路：

解法一：按照栈的思想，如果遇到的是左括号，就存入栈中，当遇到右括号时，考察右括号跟弹出栈的左括号是否对应，如果是就继续考察后面的括号，如果不是，直接返回false。

解法二、；利用了无序的hash表，存储括号的对应关系，同样利用栈的思想，在遇到右括号时，考察栈顶的是否与其对应，若是，则继续考察后面的括号，否则返回false。虽然相比较解法一而言，思想都是相同的，但是，代码量小。

实现代码：

/*解法一：*/
class Solution {
public:

21.合并两个有序链表（简单）

题目描述：

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

题解思路：

解法一：需要新建一个链表用来存储两个链表比较后要填入的值。

解法二；利用递归。

实现代码：

/*解法一：*/
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
        if(l1 == nullptr){
            return l2;
        }else if(l2 == nullptr){
            return l1;
        }

        ListNode* root = new ListNode();
        ListNode* newListNode = root;
        while(l1!=nullptr || l2 != nullptr){
            if(l1!=nullptr && l2 != nullptr && l1->val <= l2->val){
                ListNode* newVal = new ListNode(l1->val);
                newListNode->next = newVal;
                newListNode = newListNode->next;
                l1 = l1->next;
            }else if(l1!=nullptr && l2 != nullptr && l1->val > l2->val){
                ListNode* newVal = new ListNode(l2->val);
                newListNode->next = newVal;
                newListNode = newListNode->next;
                l2 = l2->next;
            }

            while(l1!=nullptr && l2 == nullptr){
                newListNode->next = l1;
                newListNode = newListNode->next;
                l1 = l1->next;
            }
            while(l1==nullptr && l2 != nullptr){
                newListNode->next = l2;
                newListNode = newListNode->next;
                l2 = l2->next;
            }
        }

        return root->next;
    }
};

解法一的复杂度分析：

  • 时间复杂度：O(m+n)，其中m和n分别是两个链表的长度；
  • 空间复杂度：O(1)，只使用了常数个变量。

53.最大子序和（简单）

题目描述：

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

题解思路：

先定义一个dp数组，其中dp[i]表示的是数组nums[i]以i为索引之前最大和的连续子数组的和，因为题目说的是子数组最少包含一个元素，所以如果dp[i-1]为负的，那么dp[i]直接取nums[i]，否则就取dp[i-1]+nums[i]。动态转移方程为:
$$
dp(i) = max(nums[i],dp[i-1]+nums[i])
$$

实现代码：

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int n = nums.size();

        vector<int> dp(n+1,0);
        int max_val = nums[0];
        dp[0] = nums[0];
        for(int i = 1; i < n; i++){
            dp[i] = max(nums[i], dp[i-1]+nums[i]);
            max_val = max(dp[i],max_val);
        }
        
        return max_val;
    }
};

代码的复杂度分析：

  • 时间复杂度：O(n)；
  • 空间复杂度：O(n)。

94.二叉树的中序遍历（简单）

题目描述：

给定一个二叉树的根节点 root ，返回它的 中序 遍历。

题解思路：

先说说二叉树的前序遍历、中序遍历以及后续遍历。

  • 前序遍历：先访问根结点，然后再访问左子结点，最后访问右子结点；
  • 中序遍历：先访问左子结点，再访问根结点，最后访问右子结点；
  • 后序遍历：先访问左子结点，再访问右子结点，最后访问根结点。

举个例子：

解法一：递归。对于中序遍历，直接利用递归找到左边的叶子结点，然后依次利用容器存储该节点的值。然后再对右子节点而言，考察其左子节点。

解法二：迭代。

拓展前序遍历和后续遍历的迭代方法代码。

//前序和后续
vector<int> preErgodic(TreeNode* root){
    vector<int> res;
    stack<TreeNode*> st;
    while(root != nullptr || !st.empty()){
        while (root != nullptr)
        {
            res.emplace_back(root->val);
            st.push(root);
            root = root->left;
        }
        root = st.top();
        st.pop();
        root = root->right;
        
    }
    return res;
}

vector<int> afterErgodic(TreeNode* root){
    vector<int> res;
    stack<TreeNode*> st;
    TreeNode* r = NULL;
    while(root != nullptr || !st.empty()){
        while(root!= nullptr){
            st.push(root);
            root = root->left;
        }
        
        root = st.top();

        if(root->right != nullptr && root->right != r){
            root = root->right;
        }else{
            st.pop();
            res.emplace_back(root->val);
            r = root;
            root = nullptr;
        }    

    }
    return res;
}

101.对称二叉树（简单）

题目描述：

给定一个二叉树，检查它是否是镜像对称的。

题解思路：

直接递归遍历即可。如果根结点不存在，返回true；如果根结点存在但没有叶子结点，也返回true；如果叶子结点存在就比较这两个叶子结点的值是否相等，如果不等就返回false，否则就继续比较left.left和right.right是否相等，以及left.right和right.left是否相等。 

实现代码：

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        if(root == nullptr){
            return true;
        }

        return dfs(root->left, root->right);
    }

    bool dfs(TreeNode* left, TreeNode* right){
        if(left == nullptr && right == nullptr)return true;
        
        if(left == nullptr || right == nullptr){
            return false;
        }

        if(left->val == right->val ){
            return dfs(left->left, right->right) && dfs(left->right, right->left);
        }else{
            return false;
        }
    }
};

代码的复杂度分析：

  • 时间复杂度：$O(n)$,其中n表示数中节点个数；
  • 空间复杂度：渐进空间复杂度为$O(n)$
//迭代
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}

class Solution {
    bool check(TreeNode* u, TreeNode* v){
        queue<TreeNode*> que;
        que.push(u);
        que.push(v);
        while(!que.empty()){
            u = que.front(); que.pop();
            v = que.front(); que.pop();
            if(u == nullptr && v == nullptr) continue;
            if((u == nullptr || v == nullptr) || (u->val != v->val)){
                return false;
            }

            que.push(u->left);
            que.push(v->right);
            que.push(u->right);
            que.push(v->left);
        }
        return true;
    }
public:

        if(root == nullptr) return true;
        return check(root->left,root->right);

};

104.二叉树的最大深度

题目描述：

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

题解思路：

解法一：深度优先搜索，利用递归去一步步搜索。

解法二：广度优先搜索，利用层遍历，依次将每一层的所有结点存储进来，然后再依次弹出队列，这样维护队列并记录维护的次数即可。

实现代码：

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
//解法一：深度优先搜索
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if(root == nullptr)
            return 0;
        return max(maxDepth(root->left), maxDepth(root->right))+1;
    }
};

解法一的复杂度分析：

  • 时间复杂度：O(n)，其中n为二叉树节点的个数，每个节点在递归中只被遍历一次；
  • 空间复杂度：O(height)，其中height表示树的高度。递归函数需要栈空间，而栈空间取决于递归的深度，因此空间复杂度等价于二叉树的高度。
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
//解法二：广度优先搜索
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if(root == nullptr)
            return 0;
        queue<TreeNode*> Q;
        Q.push(root);
        int ans = 0;
        while(!Q.empty()){
            int sz = Q.size();
            while(sz-- > 0){
                TreeNode* newNode = Q.front();
                Q.pop();
                if(newNode->left) Q.push(newNode->left);
                if(newNode->right) Q.push(newNode->right);
            }
            ans++;
        }
        return ans;
    }
};

解法二的复杂度分析：

  • 时间复杂度；O(n)，需要遍历二叉树中所有的节点；
  • 空间复杂度：取决于队列中的元素个数，最差是O(n)。

121.买卖股票的最佳时机

题目描述：

给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0

题解思路：

解法一：最容易想到的是直接暴力穷举，但是超时了。

解法二：暴力穷举的时候利用了两个for循环，如何实现只需要遍历一遍。例如假定一个dp[i]表示在第i+1天卖出能获得的最大利润，也就是在第i位置之前找比当前值最小的值。那么动态转移方程为dp[i]=max(dp[i-1], prices[i]-minV)，其中minV = min(minV, prices[i])。

解法二改进，找一个常量maxV去记录最大值。

实现代码：

//暴力穷举
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int maxV = 0;
        for(int i = 0; i < prices.size(); i++){
            for(int j = i + 1; j < prices.size(); j++){
                int res = prices[j] - prices[i];
                maxV = max(maxV, res);
            }
        }
        return maxV;
    }
};

解法一的复杂度分析：

  • 时间复杂度：O(N^2)因为使用了两个for循环；
  • 空间复杂度：O(1)使用了常数个变量。
//解法二
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        //dp[i]表示在第i+1天卖出时获得的最大利润
        //vector<int> dp(n+1,0);
        int minV = prices[0];
        int maxV = 0;
        for(int i = 1; i < n; i++){
            minV = min(minV, prices[i]);
            //dp[i] = max(dp[i-1], prices[i] - minV);
            maxV = max(maxV, prices[i] - minV);
        }
        return maxV;
    }
};

解法二的复杂度分析：

  • 时间复杂度：O(N)使用了一个for循环；
  • 空间复杂度：O(1)使用了常数个变量（改进的）。（未改进的空间复杂度是O(N)，因为有一个dp数组。）

136.只出现一次的数字（简单）

题目描述：

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

题解思路：

为了找到其中仅仅只出现一次的值（其余的出现两次），直接异或操作就可以得到该值。因为异或运算满足如下的规律：

  • $a\bigoplus0=a$
  • $a\bigoplus a=0$
  • $a\bigoplus b\bigoplus a = (a\bigoplus a) \bigoplus b = 0 \bigoplus b = b$

实现代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        if(nums.empty()){
            return 0;
        }
        
        int t = nums[0];
        for(int i = 1; i < nums.size(); i++){
            t = t ^ nums[i];
        }
        return t;
    }
};

代码的复杂度分析：

  • 时间复杂度：O(N)使用了一个for循环；
  • 空间复杂度：O(1)使用了常数个变量。

141.环形链表（简单）

题目描述：

给定一个链表，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。

如果链表中存在环，则返回 true 。 否则，返回 false 。

题解思路：

解法一：找链表中的环直接使用快慢指针，快指针每次走两步，慢指针每次只走一步，如果快指针能够追上慢指针，那么证明有环。否则，无环。 

解法二：利用hash表，记录每次遍历的节点，如果找到某个节点在hash表中存在，证明有环，否则没有环。

实现代码：

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
//解法一：快慢指针
class Solution {
public:
    bool hasCycle(ListNode *head) {
        if(head == nullptr){
            return false;
        }
        
        ListNode* fast = head;
        ListNode* slow = head;

        do{
            if(fast == nullptr || fast->next == nullptr) return false;
            fast = fast->next->next;
            slow = slow->next;
        }while(fast != slow);
        return true;

    }
};

代码的复杂度分析：

  • 时间复杂度：O(N)，其中N为链表中的节点数。当链表中不存在环的时候，快指针比慢指针优先达到链表尾部，链表中每个节点最多被访问两次。当链表中存在环的时候，每一轮移动后，快慢指针的距离将减小一。而初始距离为环的长度，因此最多移动N轮。
  • 空间复杂度：O(1)只使用了两个变量，快慢指针。

206.反转链表（简单）

题目描述：

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

题解思路：

解法一：利用递归来实现链表的反转。在解法一中有一个很核心的代码：
$$
head->next->next = head;\head->next = NULL
$$
就如$n_{k}$是链表中的第k个节点，然后要执行反转是让$n_{k+2}$变成$n_{k}$，而$n_{k+1}=NULL$。而返回的curr是头结点（反转后的头结点）。

解法二：迭代。

//解法一
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}

class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        if(head == nullptr || head->next == nullptr){
            return head;
        }
        ListNode* curr = reverseList(head->next);
        head->next->next = head;
        head->next = NULL;

        return curr;
    }
};

class Solution {
public:

class Solution {

        int count = 0;
        for(int i = low; i <= high; i++){
            if(nums[i] == target){
                count++;
            }
        }
        return count;
    }

        if(low == high){
            return nums[low];

        int mid = low + (high - low) / 2;
        int left_majority = majority_element_rec(nums, low, mid);
        int right_majority = majority_element_rec(nums, mid+1, high);

        if(countInRange(nums, left_majority, low, high) > (high - low + 1) / 2){
            return left_majority;
        }

        if(countInRange(nums, right_majority, low, high) > (high - low +1) / 2){
            return right_majority;
        }
        return -1; 
    }

    }

226.翻转二叉树（简单）

题目描述：

翻转一颗二叉树。

题解思路：

看这道题的第一眼，就想到了101对称二叉树，所以按照先前的套路写了值交换，在运行案例[1,2]时发现值交换并不会变成[1,null,2]。故还是得按照结点交换。

利用递归，在每一次递归的时候找到左节点和右节点，然后更改指向（也就是交换）。看实现代码，递归是在找到叶子结点时才开始交换，首先交换的是左边。例如示例，在找到叶结点1时，继续搜索该节点的子节点，发现都是nullptr，执行交换。但是由于1为叶子节点故没什么变化，返回节点1。开始执行节点2的右子节点，同样叶子节点3也执行了交换，返回节点3。然后节点1和节点3执行交换，变为2的左节点是3，右节点是1。返回节点2，开始执行节点4的右子节点的交换过程，最后返回节点7，此时节点7的左子结点为9，右子节点为6，然后交换节点2和节点7，返回节点4。至此，二叉树的反转完毕。

解法一：先异或操作可以得到哪些位置是不同的，然后利用移位来统计有多少个1.

解法二：布赖恩·克尼根算法。算法的思路如下：

如果是想统计$5_{d}=(101)_{b}$中1的个数，我们就让5与（5-1）按位与，过程如下：
$$
101\ \underline{& 100}\ \Downarrow \ 100 \
\underline{& 011} \\Downarrow \000
$$
从上面的过程中可以看到，5&4=4,4&3=0，其中&操作了两次，则5中1的个数为2.

实现代码：

//解法一：异或和移位
class Solution {
public:

        int xor_v = x ^ y;
        int distance = 0;
        while(xor_v != 0){
            if(xor_v&1){
                distance++;
            }
            xor_v = xor_v >> 1;
        } 
        return distance;
    }
};

  • 时间复杂度：O(1)，int的大小固定，处理时间也是固定的。32位整数需要32次迭代。

543.二叉树的直径（简单）

题目描述：

给定一棵二叉树，你需要计算它的直径长度。一棵二叉树的直径长度是任意两个结点路径长度中的最大值。这条路径可能穿过也可能不穿过根结点。

题解思路：

这道题容易让人联想到求树的深度（104），但是题目要求的是直径。可以观察对于任意一个结点，其求得的左深度和右深度之和，就是待求的所谓直径，然后记录这个直径的最大值就是整棵树的直径。

实现代码：

//
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
    int ans;
    int dfs(TreeNode* head){
        if(head == nullptr){
            return 0;
        }
        int left_depth = dfs(head->left);
        int right_depth = dfs(head->right);
        ans = max(ans, left_depth+right_depth);
        return max(left_depth,right_depth)+1;
    }

public:
    int diameterOfBinaryTree(TreeNode* root) {
        ans = 0;
        dfs(root);
        return ans;
    }

};

代码的复杂度分析：

  • 时间复杂度：O(n)，n是树的结点数，递归需要遍历所有的结点。
  • 空间复杂度：O(height)，其中height为二叉树的高度，由于递归函数在递归过程中需要为每一层递归函数分配栈空间，所以这里需要额外的空间且该空间取决于递归的深度，而递归的深度显然为二叉树的高度，并且每次递归调用的函数里面又用了常数个变量，所以需要的空间复杂度为O(Height)。

617.合并二叉树（简单）

题目描述：

给定两个二叉树，想象当你将它们中的一个覆盖到另一个上时，两个二叉树的一些节点便会重叠。

你需要将他们合并为一个新的二叉树。合并的规则是如果两个节点重叠，那么将他们的值相加作为节点合并后的新值，否则不为 NULL 的节点将直接作为新二叉树的节点。

题解思路：

解法一：深度优先搜索，从根结点开始遍历两个二叉树，并将对应的节点进行合并。两个二叉树的对应节点可能存在以下三种情况，对于每种情况的使用不同的合并方式。

  • 如果两个二叉树的对应节点为空，那么合并后的对应节点也是空；
  • 如果两个二叉树的对应节点有一个为空，另一个不为空，那么合并后的对应节点就是那个不为空的节点；
  • 如果两个二叉树的对应节点均不为空，那么合并后的节点就是两个节点值之和，此时需要显性合并两个节点。

解法一是要开辟新的节点，可以考虑直接在原树上进行操作，例如root1。参考解法二。

解法三：广度优先搜索，需要一个额外的数据结构来辅助，可以是栈也可以是队列。

首先，当两个二叉树的左子结点不为空，就放入队列中，如果两个二叉树的右子结点不为空，也把右子节点存入队列中。

如果树1的左子结点为空，而树2的左子结点不为空，就把树2的左子节点挂到树1的上面；同理如果树1的右子节点为空，而树2的右子节点不为空，就把树2的右子节点挂到树1上面。如果都不为空，就将值相加然后赋给树1的对应节点。

实现代码：

//解法一：深度优先搜索
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* mergeTrees(TreeNode* root1, TreeNode* root2) {
        if(root1 == nullptr) return root2;
        if(root2 == nullptr) return root1;
        auto newNode = new TreeNode(root1->val+root2->val);
        newNode->left = mergeTrees(root1->left, root2->left);
        newNode->right = mergeTrees(root1->right, root2->right);
        return newNode;
    }
};

解法一的复杂度分析：

  • 时间复杂度：O(min(m,n))，其中m和n分别是两个二叉树的节点树。对两个二叉树同时进行深度优先搜索，只有当两个二叉树中的对应节点不为空时才会对该节点进行显性合并操作，因此被访问到的节点树不会超过较小的二叉树的节点数；
  • 空间复杂度：O(min(m,n))，其中m和n分别是两个二叉树的节点个数。空间复杂度取决于递归调用的层数，递归调用的层数不会超过较小二叉树的最大高度，最坏的情况下，二叉树的高度等于结点数。

 * Definition for a binary tree node.

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* mergeTrees(TreeNode* root1, TreeNode* root2) {
        if(root1 == nullptr || root2 == nullptr) return root1 == nullptr ? root2:root1;

        queue<TreeNode*> q;
        q.push(root1);
        q.push(root2);

        while(q.size()>0){
            TreeNode* t1 = q.front();
            q.pop();
            TreeNode* t2 = q.front();
            q.pop();
            t1->val += t2->val;

            //如果t1和t2的左子树不为空，就放入队列中
            //如果t1的左子树为空，就把t2的左子树挂到t1的左子树上
            if(t1->left != nullptr && t2->left != nullptr){
                q.push(t1->left);
                q.push(t2->left);
            }else if(t1->left == nullptr){
                t1->left = t2->left;
            }

            //如果t1和t2的右子树不为空，就放入队列
            //如果t1的右子树为空，将t2的右子树挂到t1的右子树上去
            if(t1->right != nullptr && t2->right != nullptr){
                q.push(t1->right);
                q.push(t2->right);
            }else if(t1->right == nullptr){
                t1->right = t2->right;
            }

        }
        return root1;
    }

};

  • 时间复杂度：O(min(m,n))，其中m和n为两个二叉树的节点个数。对两个二叉树同时进行广度优先搜索，只有当两个二叉树中的对应节点都不为空的时候才会访问到该节点，因此被访问到的节点数不会超过较小的二叉树的节点数；

3.无重复字符的最长子串（中等）

题目描述：

给定一个字符串，请你找出其中不含有重复字符的最长子串的长度。

题解思路：

滑动窗口求解，并利用一个hash表来辅助，如果右指针指向的元素在hash表中没有，就把该元素拷贝到hash表中，如果右指针指向的元素在hs表中存在，那么利用左指针，将hs表中的数据清除，并将新的元素（与之前重复的元素）放入hash表中，然后利用一个maxStr用来记录其中最大的无重复子串的长度。

实现代码：

class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int n = s.length();
        if(n == 0) return 0;
        unordered_set<char> hs;
        int left = 0;
        int maxStr = 0;
        for(int i = 0; i < n; i++){
            while(hs.count(s[i])){
                hs.erase(s[left]);
                left++;
            }
            maxStr = max(maxStr, i - left + 1);
            hs.insert(s[i]);
        }
        return maxStr;
    }
};

代码的复杂度分析：

  • 时间复杂度：O(n)，n是字符串的长度，左指针和右指针分别会遍历一次整个字符长度；
  • 空间复杂度：O($|\Sigma|$)，其中$\Sigma$表示字符集

题解思路：

首先对原始的数组进行一个排序，如果发现某个元素的值大于0，就说明其后面的数字不可能相加等于0（因为都是正数）。因此当遇到nums[i]>0时就可以退出并返回结果。如果索引i处对应的元素值为非正数，再定义两个指针left和right分别为i+1和n-1，依次遍历后面的元素查看是否具有nums[i]+nums[left]+nums[right]=0,如果没有，就考虑相加的结果是大了还是小了，如果是大了，就让right减一，反之就让left加一。

在得到nums[i]+nums[left]+nums[right]=0时，就用数组将该三个值记录下来，并考察left与left+1对应的元素是否相等（主要是为了消除重复的情况），如果是相等就让left加一至于其下一位不相等为止，同样对于right进行相同的操作。

            return vector<vector<int>>{};

        vector<vector<int>> res;

            if(nums[i]<=0){
                if(i > 0 && nums[i] == nums[i-1])
                    continue;

                int left = i+1;
                int right = n - 1;
                while(left<right){
                    int tmp = nums[i]+nums[left]+nums[right];
                    if(tmp > 0){
                        right--;
                    }else if(tmp < 0){
                        left++;
                    }else{
                        res.emplace_back(vector<int>{nums[i],nums[left],nums[right]});
                        while(left<right && nums[left] == nums[left+1]){
                            left++;
                        }

                        while(left < right && nums[right] == nums[right-1]){
                            right--;
                        }
                        //这里还进行此操作是因为上面的操作没有让left指向不与其之前元素重复的值上，同样对于right。
                        left++;
                        right--;
                    }
                }
            }else{

            }
        }

        return res;
    }

这道题是在第k个位置进行翻转，可以把数组分两部分来看，其中必有一部分是有序的，因此可以使用二分查找来解决该问题。

首先取中间的数与第一个数进行比较，如果中间的数nums[mid]<nums[0]，那么右边部分[mid+1,n-1]是有序的。然后继续考察target是否在右边部分，如果是就让left=mid+1，否则就是在左边部分，就让right=mid-1。如果nums[mid]>nums[0]，就证明左半部分是有序的，然后继续考察target。

class Solution {
public:
    int search(vector<int>& nums, int target) {
        int n = nums.size();
        if(n == 0){
            return -1;
        }else if(n == 1){
            return nums[0] == target?0:-1;
        }
        int left = 0;
        int right = n - 1;
        while(left<=right){
            int mid = left + (right - left) / 2;
            if(nums[mid] == target){
                return mid;
            }else{
                if(nums[mid] >= nums[0]){
                    //前半部分有序
                    if(nums[0]<=target && target < nums[mid]){
                        right = mid - 1;
                    }else{
                        left = mid + 1;
                    }
                }else{
                    //后半部分有序
                    if(nums[mid]<target && target <=nums[n-1]){
                        left = mid + 1;
                    }else{
                        right = mid - 1;
                    }
                }
            }
        }
        return -1;
    }
};

解决这个下一个排列的问题，主要是按照以下步骤：

  • 首先从后往前遍历查找非升序的第一个元素，例如[4,5,2,6,3,1]中的2。

  • 然后再从后往前找到第一个大于2的元素，在例子中是3。

  • 交换2和3，得到[4,5,3,6,2,1]。然后对3之后的元素进行翻转得到[4,5,3,1,2,6]就是所求的下一个排列。

class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();
        int k = n - 2;
        while(k >= 0 && nums[k]>=nums[k+1]){
            k--;
        }
        if(k >= 0){
            int s = n -1;
            while(s > k && nums[s] <= nums[k]){
                s--;
            }
            swap(nums[k],nums[s]);
           
        }
        reverse(nums.begin()+k+1,nums.end());
    }
};

题目描述：

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

题解思路：

解法一：首先对这个数组进行排序，其次在排完序后考虑前一个二元数组a与后一个二元数组b边界大小比较：

  • 如果$a_{1}<b_{0}$，说明两个数组没有交界，可以直接将前一个数组存入答案中； 

  • 如果$a_{1}>=b_{0}, a_{1}<=b_{1}$，那么让$(a_{0},b_{1})$组成新的数组与后面的再进行比较。

实现代码：

//解法一：
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        int n = intervals.size();
        sort(intervals.begin(),intervals.end());

        vector<vector<int>> res;
        vector<int> tmp = intervals[0];
        for(int i = 1; i < n; i++){
            if( tmp[1] < intervals[i][0]){
                res.emplace_back(tmp);
                tmp = intervals[i];
            }else{
                if(tmp[0] <= intervals[i][0] && tmp[1] <= intervals[i][1]){
                    tmp = vector<int>{tmp[0],intervals[i][1]};
                }
            }

            if(i == n - 1){
                res.emplace_back(tmp);
            }
        }

        if(n < 2){
            res.emplace_back(tmp);
        }

        return res;

    }
};

解法一的复杂度分析：

  • 时间复杂度：O(nlogn)，n为数组的长度，先对数组进行一次排序，需要的时间为O(nlogn)，再对原始的数组遍历一次所需要的时间为O(n)，因此综合来看，最终代码整体的时间复杂度为O(nlogn)；
  • 空间复杂度：O(1)，使用了常数个变量。

64.最小路径和（中等）

题目描述：

给定一个包含非负整数的 *m* x *n* 网格 grid ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

题解思路：

解法一：动态规划。

  • 首先定义一个mn的二维数组，让$dp[0][0]=grid[0][0]$。

  • 当i等于0而j不等于0的时候，让$dp[i][j]=dp[i][j-1]+grid[i][j]$；当i不等于0而j等于0的时候，让$dp[i][j]=dp[i-1][j]+grid[i][j]$。 

  • 当i和j都不为0时，使用动态方程：$ dp[i][j]=min(dp[i-1][j],dp[i][j-1])+1$。

实现代码：

//解法一：动态规划
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        int m = grid.size();
        int n = grid[0].size();
        vector<vector<int>> dp(m,vector<int>(n));
        dp[0][0] = grid[0][0];
        for(int i = 0; i < m; i++){
            for(int j = 0; j < n; j++){
                if(i == 0 && j != 0) dp[i][j] = dp[i][j-1]+grid[i][j];
                else if(i != 0 && j == 0) dp[i][j] = dp[i-1][j]+grid[i][j];
                else if(i > 0 && j > 0){
                    dp[i][j] = min(dp[i-1][j],dp[i][j-1])+grid[i][j];
                }
            }
        }
        return dp[m-1][n-1];
    }
};

解法一的复杂度分析：

  • 时间复杂度：O(mn)，mn为数组的长度，代码中使用了两个for循环；
  • 空间复杂度：O(mn)，主要的消耗来自于dp数组。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

题解思路：

回溯。先找到第一个匹配的元素，然后在其上下左右的相邻元素是否与第二个元素匹配，如果不匹配则返回，如果四个方向都没找到匹配的元素直接返回false。如果找到匹配的元素，则继续按照之前的一步步查找。

实现代码：

//回溯
class Solution {
    bool check(vector<vector<char>>& board, vector<vector<bool>>& flags_v, int i, int j, string word, int k){
        if(board[i][j] != word[k]){
            return false;
        }else if(k == word.length() - 1){
            return true;
        }
        flags_v[i][j] = true;
        //方位数组
        vector<pair<int, int>> dirs{{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
        bool result = false;
        for(auto& dir : dirs){
            //相邻的位置
            int i_num = i + dir.first;
            int j_num = j + dir.second;
            if(i_num >= 0 && i_num < board.size() && j_num >= 0 && j_num < board[i].size() && !flags_v[i_num][j_num]){
                bool flag = check(board, flags_v, i_num, j_num, word, k + 1);
                if(flag){
                    result = true;
                    break;
                }
            }
        }
        flags_v[i][j] = false;

        return result;
    }
public:
    bool exist(vector<vector<char>>& board, string word) {
        int m = board.size();
        int n = board[0].size();

        vector<vector<bool>> flags_v(m, vector<bool>(n));

        for(int i = 0; i < m; i++){
            for(int j = 0; j < n; j++){
                if(board[i][j] == word[0]){
                    bool flag = check(board,flags_v,i,j,word,0);
                    if(flag) return true;
                }
            }
        }

        return false;
    }
};

代码的复杂度：

  • 时间复杂度：$O(mn3^L)$，其中m和n表示数组的长度，L是word的长度，除了第一个匹配的元素，后面的所有元素都是在3个方向考虑，但是由于剪枝的存在，所以时间复杂度肯定是要比$O(mn3^L)$。
  • 空间复杂度：$O(mn)$，额外开辟了一个flags_v数组，同时栈的深度为$O(min(mn,L))$。

  • boolean search(String word) 如果字符串 word 在前缀树中，返回 true（即，在检索之前已经插入）；否则，返回 false 。

  • boolean startsWith(String prefix) 如果之前已经插入的字符串 word 的前缀之一为 prefix ，返回 true ；否则，返回 false 。

题解思路：

参考：https://leetcode-cn.com/problems/implement-trie-prefix-tree/solution/trie-tree-de-shi-xian-gua-he-chu-xue-zhe-by-huwt/

class Trie {
private:
    bool isEnd;
    Trie* next[26];
public:  
    /** Initialize your data structure here. */
    Trie() {
        isEnd = false;
        memset(next,0,sizeof(next));
    }
    
    /** Inserts a word into the trie. */
    void insert(string word) {
        Trie* nodes = this;
        for(auto& c: word){
            if(nodes->next[c-'a'] == NULL){
                nodes->next[c-'a'] = new Trie();
            }
            nodes = nodes->next[c-'a'];
        }
        nodes->isEnd = true;
    }
    
    /** Returns if the word is in the trie. */
    bool search(string word) {
        Trie* nodes = this;
        for(auto& c: word){
            nodes = nodes->next[c-'a'];
            if(nodes == NULL){
                return false;
            }
        }
        return nodes->isEnd;
    }
    
    /** Returns if there is any word in the trie that starts with the given prefix. */
    bool startsWith(string prefix) {
        Trie* nodes = this;
        for(char c: prefix){
            nodes = nodes->next[c-'a'];
            if(nodes == NULL){
                return false;
            }
        }
        return true;
    }
};

236.二叉树的最近公共祖先（中等）

题目描述：

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

题解思路：

参考：https://leetcode-cn.com/problems/lowest-common-ancestor-of-a-binary-tree/solution/236-er-cha-shu-de-zui-jin-gong-gong-zu-xian-hou-xu/

实现代码：

//
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        if(!root || root == p || root == q){
            return root;
        }
        TreeNode* left = lowestCommonAncestor(root->left,p,q);
        TreeNode* right = lowestCommonAncestor(root->right,p,q);

        if(!left){
            return right;
        }

        if(!right){
            return left;
        }

        return root;
    }
};

代码的复杂度分析：

  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(n)$

238.除自身以外数组的乘积（中等）

题目描述：

给你一个长度为 n 的整数数组 nums，其中 n > 1，返回输出数组 output ，其中 output[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。

题解思路：

实现代码：

//解法一：左右乘积列表
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> L(n,1);
        vector<int> R(n,1);
        vector<int> ans(n);
        for(int i = 1; i < n; i++){
            L[i] = L[i-1]*nums[i-1];
        }

        for(int j = n - 2; j >= 0; j--){
            R[j] = R[j + 1] * nums[j + 1];
        }

        for(int k = 0; k < n; k++){
            ans[k] = L[k]*R[k];
        }
        return ans;        
    }
};

  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(n)$

240.搜索二维矩阵II（中等）

题目描述：

编写一个高效的算法来搜索 *m* x *n* 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：

  • 每行的元素从左到右升序排列；

  • 每列的元素从上到下升序排列。

题解思路：

解法一：我们设置开始位置为右上角，从此处开始遍历，如果当前值比目标值要大，就让col-1，否则就是让row+1，如果找到目标值就返回true，否则就根据$col<0|row>=m$跳出循环即可。

实现代码：

//解法一：
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();

        //定义初始的位置(右上角)
        int row = 0, col = n - 1;
        bool flag = false;
        while(true){
            if(matrix[row][col] == target){
                flag = true;
                break;
            }else if(matrix[row][col] > target){
                col--;
            }else{
                row++;
            }

            if(col < 0 || row >= m){
                break;
            }
        }
        return flag;
    }
};

解法一的复杂度分析：

  • 时间复杂度：$O(mn)$
  • 空间复杂度：$O(1)$
//解法二：二分查找
class Solution {
    bool binarySearch(vector<vector<int>>& matrix, int target, int low, bool vertical){
        int lo = low;
        int hi = vertical ? matrix.size() - 1: matrix[0].size() - 1;

        while(lo <= hi){
            int mid = lo + (hi - lo) / 2;
            if(!vertical){
                if(matrix[low][mid] == target){
                    return true;
                }else if (matrix[low][mid] < target){
                    lo = mid + 1;
                }else{
                    hi = mid - 1;
                }
            }else{
                if(matrix[mid][low] == target){
                    return true;
                }else if (matrix[mid][low] < target){
                    lo = mid + 1;
                }else{
                    hi = mid - 1;
                }
            }
        }

        return false;
    }
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
       int m = matrix.size();
       int n = matrix[0].size();
       int epoch = min(m,n);
       for(int i = 0; i < epoch; i++){
           bool verticalFound = binarySearch(matrix, target, i, true);
           bool horizontalFound = binarySearch(matrix, target, i ,false);

           if(verticalFound || horizontalFound){
               return true;
           }
       }

       return false;
    }
};

解法二的复杂度分析：

  • 时间复杂度：$O(min(m,n)*(log(n!)+log(m!)))$
  • 空间复杂度：$O(1)$
